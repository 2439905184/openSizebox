 # 关于此游戏
 此游戏借鉴了Sizebox沙盒游戏（巨大娘）沙盒游戏。<br>
 但是Sizebox的代码不是公开开源的，所以我仿照它做一个开源版本。<br>
 由于Sizebox是不开源的，所以我的游戏不能导入.gts文件，你可以用blender转换成gltf dae 或者vrm格式 （尽量用开源格式而不是fbx,Godot的fbx导入有问题）
